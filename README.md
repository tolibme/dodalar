# DODALAR | Food as Cultural Memory

This project is a group project for the `ILC 2250` course.

## Project Theme

**Food as Cultural Memory: Preserving Identity Through Immigration**

This website presents the Monday section of the team project. It examines how immigrant communities use traditional food practices and language to preserve cultural memory and identity in a new country.

## Team

- `DODALAR`
- Also known as `Food Hunters`

## Project Focus

The design and content explore:

- food as a carrier of cultural memory
- traditional recipes as a way to preserve identity
- language preservation through dish names, cooking instructions, and family rituals
- immigrant communities adapting to new environments without losing cultural meaning
